## Overview

Secure Contact Management Dashboard is a full-stack web application built with Node.js, Express and PostgreSQL.

The application allows visitors to submit contact messages through a public contact form while administrators can securely manage messages through a protected dashboard.

---

## Features

- Public contact form
- Session-based authentication
- Protected admin dashboard
- Message management
- Search functionality
- Status tracking
- Archive functionality
- PostgreSQL database integration
- Input validation
- Security middleware
- Rate limiting
- Responsive design
